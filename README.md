# ipl-score-forecasting-arima
# IPL Score Forecasting using ARIMA

## Project Overview
This project builds and validates an ARIMA time-series model 
to forecast IPL average scores across seasons.

## Dataset
IPL match data containing team scores across seasons.

## Methods Used
- Stationarity testing (ADF Test)
- ARIMA model building and selection
- Model diagnostics
- Future score prediction with confidence intervals

## Tools & Libraries
- Python
- pandas
- statsmodels
- matplotlib

## Results
Forecasted average IPL scores with confidence intervals 
for upcoming seasons.
